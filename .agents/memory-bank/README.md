# V-Legal Memory Bank

This folder tracks the working memory for the V-Legal prototype.

## Files

- `project-brief.md` - product intent and current scope
- `architecture.md` - current implementation structure
- `decisions.md` - high-signal build decisions and why they were made
- `progress.md` - dated progress log
- `roadmap.md` - next milestones after the current prototype

## Current Focus

Build a prototype on top of the Hugging Face dataset first, then replace or enrich weak areas with official-source pipelines from Phap dien, VBPL, and Van ban Chinh phu.

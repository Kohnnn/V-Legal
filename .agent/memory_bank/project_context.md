# Project Context: V-Legal

## High-Level Goals
Build a modern Vietnamese legal research product that is easier to trust, easier to search, and easier to extend than legacy portals.

## User Stories
- **Searchable Corpus**: Users should be able to search the local legal corpus efficiently.
- **Document Reading**: Users should have a strong archive index and gazette-style document pages for reading.
- **Metadata Filtering**: Users should be able to filter legal documents by various metadata.
- **Grounded Answers**: The system should provide grounded answer briefs that cite retrieved legal passages.
- **Law Tracking**: Users can track personal laws and receive updates.
- **Official Taxonomy**: Display the official subject taxonomy from Phap dien.
- **Relationship Graph**: Visualize relationships between documents (amended, replaced, guided).
- **Citations**: Resolve and display cross-document citations clearly.
- **Research Workbench**: Save research views and receive alerts for monitored laws.
- **Compare Mode**: Side-by-side legal document cross-checking.

## Prototype Success Criteria
- Ingestion path that can later be swapped from HF corpus to official sources.
- Retrieval-first baseline before building official crawlers.
- Validation of schema, filters, and document detail design.
